# IMDB-movie-feature-predictions
Google Colab notebook performing a data analysis on a Kaggle dataset, identifying key questions and using traditional learning techniques to identify feature importance and correlation

## Goal
The goal of this project is to analyse the higher-rated movies and identify whether different features demonstrate a positive correlation relating to performance, such as run time and year. Further ananlysis was deployed to identify the key features that yield higher-rated movies.

## Tools
 - Python
 - Pandas & NumPy
 - Matplotlib / Seaborn
 - Sklearn
 - Google Colab
